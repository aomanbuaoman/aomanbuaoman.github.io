1.3列表标签
1.1无序列表

<ul><li></li><ul>

type属性：

  • disc // 圆点
  • square //正方形
  • circle //空心圆
  • decimal //数字
  • none //无标记符合
1.2有序列表

<ol><li></li></ol>

type属性值：

  • 1 //数字1，2…
  • a //小写字母a,b…
  • A //大写字母A,B…
  • i //小写罗马数字i
  • Ⅰ //大写罗马数字Ⅰ
1.3定义列表

<dl><dt></dt><dd></dd></dl> //dt&dd为同级标签

1.4图像标签

<img src="" .../>

img属性：

  • src //图像的URL
  • alt //图像无法加载是显示的文字
  • height //高
  • width //宽
1.5超链接标签

<a href="" ></a> //内部地址&外部链接

属性：

  • href //链接

    锚点 //#+name/id 来跳转至元素

  • target //跳转界面

    self //本页跳转

    blank //新窗口

    top //清除框架载入窗口

    parent //载入父窗口或者包含来超链接引用的框架的框架集

1.6表格标签
  • table //表格标签
  • caption //表格标题
  • tr //行标签
  • th //列头标签
  • td //列标签

属性：

  • cellpadding //内容与其单元格的间距
  • cellspacing //单元格之间的距离
  • bgcolor //背景颜色

frame //外边框设置

  • above //只显示顶部
  • below //只显示底部
  • vsides //只显示左右
  • hsides //只显示上下
  • ihs //只显示左侧
  • rhs //只显示右侧
  • void //显示四周边框线
  • box/border

rules //单元格边框

  • none //不显示单元格边框
  • all //显示所有单元格边框线
  • cols //显示纵向及单元格左右
  • rows //显示横向

groups //分组显示，只显示thead/tbody/tfoot的边框线

rowspan //跨列

  • number //合并几列

colspan //跨行

  • number //合并几列
1.7表单标签

<form></form> //表单标签

属性：

  • action //提交信息的url
  • method //提交方式

get //默认

post

  • enctype //提交类型

enctype=”application/x-www-form-urlencoded”

enctype=”multipart/form-data”

  • target //在何处打开目标URL
  • name/id //名字

<input></input> //定义输入字段

type //表单项类型

  • text //单行文本框
  • password //密码专属
  • checkbox //多选框 要提供value
  • radio //单选框 要提供value
  • file //文件上传选择框
  • button //普通按钮
  • submit //提交按钮
  • image //图片提交按钮
  • reset //重置按钮，还原到开始(第一次打开页面时)
  • hidden //主表单隐藏域，一起提交，但无需修改

name //表单项名，存储内容

value //输入的值

size //输入框宽度

maxlength //输入框输入内容的最大长度

readonly //对输入框只读

disabled //禁用属性

checked 对选择框指定默认选项

accesskey //指定快捷键

tabindex //通过数字指定tab键切换数据

<select></select> //创建下拉列表

属性：

  • name //定义名称，存储下拉值
  • size //定义可见项目
  • disabled //当该属性为true时，禁用该菜单
  • multiple //多选

<option></option> //下拉时选择项的标签，嵌套到input中

  • value //下拉项的值
  • selected //默认下拉指定项

<textarea> //多行文本输入区域

  • name //定义名称，存储值
  • cols //文本区可见列数
  • rows //文本区可见行数
  • disabled //是否禁用
  • readonly //只读

<button></button> //标签定义按钮，与input不同之处：可放置文档或图像

<fieldset></fieldset> //将表单内元素分组

  • disabled //定义fieldset是否可见
  • form //所属一个或多个表单

<optgroup> //定义选项组，在选项中分组

<datalist> //定义可输入筛选列表可选数据的列表
